Request URL Response
GET /movies Return a list holding data about all movies
GET /movies/:title Return data about one movie, containing description, genre, direction, and image URL
GET /movies/:genre Return data about one movie's genre
GET director/:name Return data about one movie's director
POST /register Return data about a new user who registered
PUT /users/:id/:user_info Text message indicating user info was updated
POST users/:id/:favorites Text message listing movies added to favorites
DELETE /users/:id/favorites Text message indicating a movie was removed from favorites
DELETE /users/:id/unregister Text message telling a user their account is deregistered
